12 — Q&A

Common Questions

Q: What architecture did you use?
Client-Server Architecture. Next.js frontend communicates with Python FastAPI backend via REST API using Axios HTTP client.
Q: What is Glassmorphism?
A CSS design trend — creates frosted glass effect using backdrop-filter: blur(), semi-transparent backgrounds, and subtle borders.
Q: How does the AI Recommender work?
Frontend sends user preferences via POST to FastAPI backend. Backend analyzes keywords and recommends a design style (Minimalist, Scandinavian, Industrial, or Modern Contemporary).
Q: How does the Before/After slider work?
Two images are stacked. A draggable slider controls CSS clip-path that reveals/hides portions of each image. Supports both mouse and touch events.
Q: What is CORS and why is it needed?
Cross-Origin Resource Sharing — needed because frontend (port 3000) and backend (port 8000) run on different origins. FastAPI CORS middleware allows cross-origin API requests.
Q: What design patterns did you use?
Component-based architecture (React), MVC pattern (backend), ORM pattern (SQLAlchemy), Repository pattern (session management), and REST API design.
